# TastyBlog - Кулинарный блог

## Описание проекта
TastyBlog — веб-приложение для публикации кулинарных рецептов с современным дизайном и удобным интерфейсом.

## Функционал
- Просмотр рецептов с фотографиями
- Поиск по категориям
- Адаптивный дизайн
- Система оценок

##  Технологии
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Tools: Git, npm, Prettier, Figma
- Design: Figma для прототипирования

##  Дизайн
