# TastyBlog - Кулинарный блог

## Описание проекта

TastyBlog – веб-приложение для публикации кулинарных рецептов с современным дизайном и удобным интерфейсом.

## Функционал

- Просмотр рецептов с фотографиями
- Поиск по категориям
- Адаптивный дизайн
- Система оценок и комментариев
- Фильтрация по времени приготовления
- Сохранение любимых рецептов
- Пошаговые инструкции приготовления

## Технологии

- HTML5
- CSS3
- JavaScript
- Figma (дизайн)
- Git (контроль версий)
- Prettier (форматирование кода)
